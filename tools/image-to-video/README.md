# Image to Video

Animate static images into motion videos using AI. Transform photos into cinematic, anime, 3D render, or motion graphic videos.

## Features

- Drag-and-drop image upload
- Motion prompt input (e.g., "slow zoom in", "gentle sway")
- 5 style presets: Cinematic, Anime, 3D Render, Realistic, Motion Graphics
- Adjustable duration (2-10 seconds)
- Multiple FPS options (24/30/60)
- Real-time progress tracking
- MP4 download and canvas integration

## How to Run

Open `src/index.html` in a web browser.

## Workflow

1. Upload an image (JPG, PNG, WebP)
2. Write motion description (optional but recommended)
3. Select style and duration
4. Choose FPS (24/30/60)
5. Click "Generate Video"
6. Download MP4 or send to canvas

## Tips

- Cinematic preset works for most use cases
- Motion prompts are key: be descriptive
- 30fps is a good balance between quality and file size
- 4-6 seconds duration recommended
