# Image to 3D

Convert 2D images into interactive 3D models with AI-powered processing. Perfect for product visualization and game assets.

## Features

- Drag-and-drop image upload
- Optional motion description prompts
- Three output formats: GLB, OBJ, FBX
- Quality levels: Fast (preview), Medium, High (slow)
- Real-time 3D preview with orbit controls
- Auto-rotate and grid helper
- Downloadable 3D model files

## How to Run

Open `src/index.html` in a web browser. Requires Three.js (loaded from CDN).

## Workflow

1. Upload a 2D image (JPG, PNG, WebP)
2. (Optional) Add a description prompt
3. Choose output format and quality
4. Click "Generate 3D Model"
5. Preview in interactive 3D viewer
6. Download GLB/OBJ/FBX

## Tips

- Clear, well-lit product photos work best
- Simple backgrounds improve conversion quality
- GLB format recommended for web/game engines
