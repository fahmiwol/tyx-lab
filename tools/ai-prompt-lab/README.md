# AI Prompt Lab

Craft, test, and refine AI image generation prompts. Write prompts, choose models and styles, generate and compare results.

## Features

- Positive prompt editor with syntax highlighting
- Negative prompt editor (what to avoid)
- Model selection (DALL-E, Stable Diffusion, etc.)
- Style presets (Photorealistic, Oil Painting, 3D Render, etc.)
- Real-time image generation
- Prompt history and bookmarks
- Batch generation
- Export prompts and images

## How to Run

Open `src/index.html` in a web browser.

## Workflow

1. Write positive prompt (what you want to see)
2. Add negative prompt (what to avoid)
3. Select model and style
4. Click "Generate"
5. View result and iterate
6. Save successful prompts

## Tips

- Use descriptive keywords: "cinematic lighting, 4k, professional photography"
- Negative prompts: "blurry, low quality, artifacts"
- Experiment with styles: portraits benefit from "photorealistic"
- Save effective prompts for reuse
