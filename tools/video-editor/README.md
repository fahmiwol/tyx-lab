# Video Editor

Non-linear video editing tool with timeline, trimming, text overlays, and audio controls.

## Features

- Multi-track timeline (video, audio, text)
- Trim and cut segments
- Text overlay with position controls
- Audio mixing panel
- Real-time preview
- Project save/load
- MP4 export

## How to Run

Open `src/index.html` in a web browser.

## Workflow

1. Import video file
2. Select tool from left panel (Select, Trim, Text, Merge, Audio)
3. Adjust properties in right panel
4. Use timeline for precise editing
5. Preview and export

## Tips

- Use trim tool to cut unwanted sections
- Text overlay supports custom positions
- Work with common formats: MP4, WebM, MOV
