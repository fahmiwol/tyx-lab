💡 Tips BG Remover

  • Gambar dengan kontras tinggi antara subjek dan background menghasilkan hasil terbaik
  • Model BiRefNet sangat baik untuk rambut, bulu, dan detail halus
  • Gambar produk e-commerce (background putih) hampir selalu sempurna
  • Untuk microstock, hasil PNG transparan langsung bisa diupload ke Dreamstime/Adobe Stock
  • Gunakan batch mode untuk proses banyak gambar sekaligus — cocok untuk pipeline otomatis
  • rembg server harus berjalan di port 9799 — jalankan server/tools/start_rembg.bat
