# NPC Generator

Generate game characters with customizable stats, roles, and rarity levels. Perfect for RPGs and world-building.

## Features

- Character roster management
- Rarity levels: Common, Rare, Epic, Legendary
- Customizable stats and attributes
- Role and skill assignment
- Character card preview
- Export character data
- Batch generation

## How to Run

Open `src/index.html` in a web browser.

## Workflow

1. Select or create a new character
2. Assign name, role, rarity, stats
3. View card preview
4. Export as JSON or image
5. Integrate into game world

## Tips

- Legendary characters have higher stat caps
- Role presets include Warrior, Mage, Rogue, Cleric
- Export JSON for game engine integration
