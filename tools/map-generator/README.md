# Map Generator

Procedurally generate game maps from descriptions. Create cyberpunk offices, nature retreats, sci-fi stations, or fantasy castles.

## Features

- Describe your world in natural language
- 4 style templates: Cyberpunk, Nature, Sci-Fi, Fantasy
- Adjustable grid size: 16×16, 24×24, 32×32
- Tile-based rendering with grid overlay
- Room and object generation
- Export as JSON for game engines
- Visual preview with emoji objects

## How to Run

Open `src/index.html` in a web browser.

## Workflow

1. Write a description (e.g., "cyberpunk office with 3 floors, server room, rooftop garden")
2. Choose grid size and style
3. Click "Generate Map"
4. Preview in isometric grid view
5. Export as JSON or send to world builder

## Tips

- More detailed descriptions = better maps
- Start with small grids (16×16) for testing
- Export JSON for integration with game engines
